# PNR Status Tracker

## Project Overview
This is a Java application that tracks PNR status using the IRCTC RapidAPI.
It is packaged as a self-contained Windows Executable (.exe).

## Locations
- **Source Code**: `src/` (Java files)
- **Executable**: open PNRtracker (Run this)

## How to Run.
2.  Open `PNRStatusTracker` 

## Developer Info
- Main Logic: `src/PNRTracker.java`
- API Logic: `src/PNRService.java`
- To modify: You need a JDK installed.
